About FTC Benelux Map

FTC Benelux Map is an unofficial tool inspired by frcmap that aims to visualize the location spread of all active FIRSTÂ® Tech Challenge teams and events in a particular season. Currently, the map only displays data from teams and events with the Benelux (officially Netherlands/NL) as their home region. More regions may be added in the future if there is sufficient demand. If you would like to see your region added, please open an issue on Github.

About FIRST

FIRST is an international nonprofit organization that prepares young people for the future through a suite of life-changing youth robotics programs that build skills, confidence, and resilience. FIRST is backed by a global network of mentors, educators, volunteers, sponsors, parents, alumni, and program delivery partners.

FIRST Tech Challenge students work together with their mentors to design and build robots to compete in a dynamic and exciting challenge released every September. Teams program classroom-sized robots to follow autonomous commands before student drivers take control in two-on-two matches. On and off the field, students develop STEM skills, engage in community outreach, practice engineering innovation, and build confidence to help them succeed.

More on firstinspires.org.

Location Data

All data is gathered directly from a publicly available API provided by FIRST and geocoded using OpenStreetMap. No team data is stored on the server that was not provided by the FIRST API, OpenStreetMap or teams themselves.

Geolocation is done through a combination of full team name, city, province and country and as such may not be 100% accurate. This is especially true for community teams. If you notice any inaccuracies about the location of an event, your own community team or any team affilliated with a public organisation, please open an issue on Github.
